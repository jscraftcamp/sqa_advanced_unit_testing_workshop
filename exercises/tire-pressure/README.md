# TDD Starter (TypeScript)

Minimal example for test-driven development: a small domain (`Alarm`) and **one passing example test** (`Alarm.test.ts`).

## Prerequisites

- **Node.js** (LTS)
- **npm**

## Stack

Tests with **Vitest**, TypeScript.

## Setup and Tests

```bash
npm install
npm test
```

Expected result: one passing test.

## Watch Mode (optional)

Re-run tests on file changes:

```bash
npm run test:watch
```
